📚 Book Management System
Tech Stack: React.js, Node.js, Express.js, MongoDB, JavaScript, HTML, CSS, REST API, Git, GitHub, Render
Developed a responsive Book Management System using React.js with complete CRUD functionality (Create, Read, Update, Delete)
Integrated RESTful APIs for seamless data fetching and state management
Implemented search functionality to filter books by title and author
Added genre-based filtering to improve data accessibility and user experience
Implemented pagination to efficiently handle and display large sets of book data
Managed loading states and error handling for better UI reliability
Built a modular and reusable component-based architecture in React
Built RESTful APIs using Node.js, Express.js, and MongoDB for handling backend CRUD operations
Deployed the application on Render for live access
